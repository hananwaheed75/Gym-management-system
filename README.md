#  Gym Management System

A web-based Gym Management System developed using PHP and MySQL with a responsive Bootstrap interface. This system helps gym administrators manage members efficiently through a simple and user-friendly dashboard.

## Features

The system includes the following core functionalities:

- Home Dashboard  
  Displays overall system overview and quick access to features

- Add Member  
  Add new gym members with complete personal and membership details

- Find Member  
  Search and view existing member records easily

- Logout  
  Secure user logout to protect system access

## Technologies Used

- HTML  
- CSS  
- Bootstrap  
- JavaScript  
- PHP  
- MySQL
